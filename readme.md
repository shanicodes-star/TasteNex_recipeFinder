# TasteNex - Recipe Finder 

TasteNex is a modern, fast, and user-friendly recipe finder platform where users can instantly search, explore, and filter thousands of recipes. From breakfast, lunch, dinner, desserts, to quick snacks, TasteNex provides complete and verified recipes for all tastes and skill levels. Its interface is simple, clean, and fast, making it ideal for beginners as well as expert cooks.

---

## Features

- Search recipes by **name, category, or cuisine**.
- Dynamic recipe cards with images, titles, and clickable buttons.
- Recipe details page showing ingredients, preparation steps, and cooking instructions.
- Filter results by recipe category or cuisine type.
- Fully responsive and mobile-friendly layout.
- Hero section with background image and search feature.
- Our Story section describing TasteNex’s mission.
- About section detailing TasteNex and Exyton Solutions.
- Blogs section dynamically loaded from JSON.
- Footer included consistently across all pages.
